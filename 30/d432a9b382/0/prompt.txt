**deintroverter4clj** is a static analyzer for Clojure/Speclj test suites. It classifies each test by whether its assertions actually reach the system under test (SUT).

**What it does:** Walks test bodies and traces asserted expressions through call chains, `let` bindings, threading macros, and var reads to determine if any assertion is grounded in production code behavior. It answers: *does this test actually verify the SUT, or is it testing thin air?*

**Input:** Clojure spec/test files (`.clj`/`.cljs`/`.cljc`). Optionally a project root for `deps.edn` discovery to infer which namespaces are SUT vs test vs external.

**Output:** Per-test verdicts — `:extroverted` (proven SUT reach), `:likely-extroverted` (heuristic), `:cloistered` (reaches test helpers but not SUT directly), `:introverted` (no SUT reach), `:questionable` (can't decide). Human-readable or EDN for agent consumption.
